# Ichime

Нативное приложение для сайта Anime 365, созданное специально для Apple TV.

Больше скриншотов можно найти на странице [Wiki → Скриншоты](https://github.com/midori-no-me/ichime/wiki/Скриншоты).

## Возможности приложения

Само приложение полностью бесплатное, но для просмотра сериалов на вашем аккаунте Anime 365 должна быть активна платная подписка.

Приложение покрывает все базовые сценарии пользования сайтом.

Просмотр серий:

- Все серии открываются в сторонних плеерах автоматически. На данный момент мы поддерживаем только [Infuse](https://apps.apple.com/app/id1136220934) и [VLC](https://apps.apple.com/app/id650377962). Эти плееры поддерживают субтитры в форматах ASS и SRT.
- Если досмотреть серию до конца, то она автоматически будет отмечена просмотренной. Так же, как это сделано на сайте.
- Выводим список следующих серий к просмотру так же, как на сайте они выводятся в секции "Серии к просмотру".

Каталог сериалов:

- Список онгоингов.
- Список тайтлов, сгруппированный по сезонам.
- Поиск.
- Показываем, когда выйдет следующая серия.
- Календарь онгоингов, который подтягивается из Shikimori.

Для получения информации о сериале мы используем разные источники, включая Shikimori и MyAnimeList (Jikan):

- Базовая информация о сериале, доступная на Anime 365.
- Название серий и их рейтинг.
- Моменты.
- Скриншоты сериала.
- Разные обложки.
- Список персонажей и сейю.
- Список авторов.
- Связанные сериалы (сиквелы, спин-оффы и т.п.).

Top Shelf:

- Если закрепить Ichime в верхнем ряду на домашнем экране Apple TV, то будем показывать в виджете секцию "Серии к просмотру" и расписание ближайших релизов.

## Как установить

По понятным причинам приложения нет в App Store.

Рекомендуемым способом установки является сервис appdb. Для первоначальной настройки потребуется Windows или macOS, а для последующих установок и обновлений приложения потребуется только iPhone или iPad. У Ichime есть официальная страница в appdb: https://appdb.to/details/6268d688903a7f61de5fe6baacb33fa880fe2d36. Руководство по установке приложений с appdb см. в статье "[How to install apps?](https://appdb.to/my/faq)".

## Поддерживается ли iOS или iPadOS?

iOS и iPadOS больше не поддерживается. Но можно скачать и установить старую версию [1.7.7](https://github.com/midori-no-me/ichime/releases/tag/1.7.7).

## Разработка приложения

### Как собрать приложение

Проект Xcode декларативно описан с помощью Tuist. Сперва необходимо [установить Tuist](https://tuist.dev/en/docs/guides/install-tuist).

Теперь можно сгенерировать проект Xcode:

```bash
tuist install
```

```bash
tuist generate
```

Для установки приложения на реальные устройства потребуется подпись приложения. Для этого в настройках проекта Xcode, во вкладке Signing & Capabilities, нужно указать свою команду. Команду можно указать вручную, но удобнее, если она будет указываться автоматически при генерации проекта. Для этого в корне проекта выполните:

```bash
cp .env.example .env
```

Затем откройте файл `.env` и отредактируйте значение переменной `TUIST_DEVELOPMENT_TEAM`.

С помощью этой команды можно перегенерировать проект, переменные из `.env` будут учитываться при генерации:

```bash
make gen
```
